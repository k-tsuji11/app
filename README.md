# バイト給料計算アプリ

バイトのシフトや時給を入力するだけで、簡単に給料の概算を計算できるWebアプリです。
「今月あといくら稼げるか？」「次の給料日はいくらか？」を素早く確認できます。

## 🌟 主な機能
* **給料シミュレーション**: 時給と勤務時間を入力して合計金額を自動計算
* **シンプル操作**: 複雑な設定なしで、誰でもすぐに使い始められます

## 🛠 使用言語
* HTML
* Python

## 📖 使い方
1. [アプリURL](https://k-tsuji11.github.io/app/) を開きます。
2. 時給を設定し、働いた時間（またはシフト）を入力します。
3. 「計算」ボタンを押すと、合計金額が表示されます。

## 開発の背景
「バイト代がいくらになるか、もっと手軽に計算したい」という悩みから作成しました。
